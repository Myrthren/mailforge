Setup & Connect

Connect Gmail
in 3 steps

MailForge uses the Gmail API via Google OAuth to send emails directly from your account. Follow the steps below — it takes about 5 minutes and only needs to be done once.

1
Create a Google Cloud Project

Go to console.cloud.google.com → Create a new project → Name it anything (e.g. "MailForge"). Then navigate to APIs & Services → Library and enable the Gmail API.

↗ Open Google Cloud Console
2
Create OAuth 2.0 Credentials

In your project go to APIs & Services → Credentials → Create Credentials → OAuth Client ID. Set Application type to Web application. Add this exact URL to Authorised JavaScript origins:

Also add the same URL to Authorised redirect URIs. Then copy your Client ID and paste it below.

3
Authorise MailForge

Once your Client ID is saved, click the button below to sign in with Google and grant MailForge permission to send emails on your behalf. You'll see a Google consent screen — this is safe and only grants access to send mail.

✓ Authorised successfully! You can now send emails.
